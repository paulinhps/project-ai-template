# Create Skill Authoring Standard

## Prompt

Create a canonical `.ai` skill named `skill-authoring-standard`.

The skill must guide AI agents such as Codex, Claude Code, Gemini CLI, Cursor, Windsurf, OpenCode, Aider, and similar tools in creating, evolving, reviewing, and refactoring skills compatible with the canonical AI ecosystem.

The skill must define consistent structure, detail level, terminology, auditability, evolution capability, overlay compatibility, future extension compatibility, low coupling, and high reuse. It must apply to canonical skills, project overlays, and any other skill definition locations.

## Result

Added `.ai/skills/skill-authoring-standard/SKILL.md` and `agents/openai.yaml`.

The skill defines mandatory sections, naming rules, dependency and priority handling, overlay compatibility, decision criteria, prohibited anti-patterns, practical examples, validation checklist, existing-skill review instructions, new-skill creation instructions, restrictions, and control rules for non-propagated operational content.
